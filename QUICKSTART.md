## Troubleshooting ğŸ”§

**App won't build?**
- Check Xcode version (14.0+ required)
- Clean build: Cmd+Shift+K
- Restart Xcode

**AI not working?**
- Add API keys in Settings
- Check network connection
- Verify API key validity

**Files not loading?**
- Check file permissions
- Restart app
- Check file path
